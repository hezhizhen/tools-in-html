# Tools in HTML

一系列单文件 HTML 工具集合。

**在线访问**: https://hezhizhen.github.io/tools-in-html/

## 工具列表

- [Base64 编解码](base64.html)

## 添加新工具

1. 在根目录创建 `工具名.html` 文件
2. 在 `index.html` 的 `<ul class="tools-list">` 中添加链接：
   ```html
   <li><a href="工具名.html">工具描述</a></li>
   ```
3. 更新本 README 的工具列表

## 启用 GitHub Pages

1. 进入仓库 Settings → Pages
2. Source 选择 `master` 分支，目录选择 `/ (root)`
3. 保存后等待部署完成
